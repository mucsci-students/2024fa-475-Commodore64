# 2024fa-475-Commodore64

Group Project by Tommy Nguyen, Valentino Pirocchi, Francesco Spagnolo

Controls:
WASD - Move
Q - fire warp projectile(in direction of mouse) / teleport to it if already fired
F - Open Inventory
Esc - Pause Shortcut
